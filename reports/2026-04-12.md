## 💡 行业观点

### 你的框架，你的记忆（Your harness, your memory）

LangChain 官方博客发文指出，Agent harness（智能体运行框架）正在成为构建 Agent 的主流方式，而这些框架与 Agent 的记忆系统深度绑定。文章的核心论点很直白：如果你选择了一个闭源框架——尤其是藏在私有 API 后面的那种——你实际上是把 Agent 记忆的控制权拱手让人。这意味着你的 Agent 积累的经验、上下文和学习成果，都锁死在别人的平台上。LangChain 显然在为自己的开源定位做背书，但这个观点本身值得每个 Agent 开发者认真思考：框架可以换，记忆丢了就真没了。

### 那个"爱马仕"，想拯救"智障"小龙虾

这篇文章以 Nous Research 发布的 Hermes Agent 为切入点，抛出了一个犀利的反思：Agent 领域最火的叙事——Skill 自动生成与持续进化——真的是当前落地的主要瓶颈吗？作者以 Claude Code 为例指出，它好用的基石不是 Skill 进化，而是背后 GlobTool、GrepTool、LSPTool 等一系列确定性的 CLI 原子操作。文章进一步拿 OpenClaw（"龙虾"）的痛点来佐证：token 消耗大、长时间工作不稳定，这些问题靠 Skill 自主进化根本解决不了。核心观点一句话概括：CLI 不性感，但它才是 Agent 能力的真正地基，地基不牢，Skill 再会长也只是长在沙地上。对于当前 Agent 开发热潮中的"叙事泡沫"，这是一剂清醒剂。

### 拿下豪华种子轮，一家明星AI公司宣布倒闭

AI 模型评测平台 Yupp 宣布将于 4 月 15 日正式关闭，从成立到关门仅 22 个月。这家公司曾在 2024 年拿下 3300 万美元（约 2.2 亿元）种子轮，背后站着 a16z 合伙人、Google 首席科学家、Twitter 联合创始人等超 45 位天使投资人。Yupp 的商业模式是通过免费模型服务吸引用户做 A/B 评测，再将测评数据卖给模型厂商。然而最荒诞的是，投资人的钱还没花完，AI 技术的快速演进就让其市场直接消失了。这个案例是 AI 赛道"窗口期极短"的典型注脚——当底层技术以月为单位迭代时，建立在某个技术阶段性缺陷上的商业模式，随时可能被技术本身的进步抹掉。

### Claude新模型引发恐慌，鲍威尔召集华尔街紧急开会，全美安全股暴跌2万亿

## 🔧 开源项目

### FlashAttention（FA1–FA4）PyTorch 教育实现：聚焦算法差异（FlashAttention (FA1–FA4) in PyTorch - educational implementations focused on algorithmic differences）

一位开发者更新了 FlashAttention-PyTorch 仓库，用纯 PyTorch 实现了 FA1 到 FA4 四个版本的教育性代码。项目的目标不是高性能 kernel，而是让开发者从代码层面理解各版本之间的算法演进：FA1 展示分块在线 softmax 基线，FA2 引入 split-Q 和延迟归一化，FA3 加入显式流水线与 ping-pong tile buffer 以及简化的 FP8 前向路径，FA4 则实现了带主阶段/softmax/校正阶段的显式调度器。对于想深入理解 FlashAttention 系列但不想一上来就啃 CUDA/Hopper/Blackwell 细节的研究者和工程师来说，这是一份非常友好的学习材料。
