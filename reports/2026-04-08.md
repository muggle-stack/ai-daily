## 🚀 模型发布

### Anthropic 的 Project Glasswing 计划——将 Claude Mythos 限制性开放给安全研究人员（Anthropic's Project Glasswing - restricting Claude Mythos to security researchers - sounds necessary to me）

Anthropic 今天没有公开发布其最新模型 Claude Mythos，而是通过全新的 Project Glasswing 计划将其限制性地提供给一小批预览合作伙伴。该模型是与 Claude Opus 4.6 类似的通用模型，但 Anthropic 声称其网络安全研究能力强大到需要给整个软件行业留出准备时间。Mythos Preview 已经发现了数千个高严重性漏洞，涵盖所有主流操作系统和浏览器。Anthropic 的判断是，随着 AI 能力快速扩散，这类能力很快会被不负责任的行为者获取，因此选择了"先防御、再开放"的策略。这是大模型厂商首次因安全能力过强而主动限制发布，标志着 AI 安全治理进入了新阶段。

## 🔧 开源项目

### Deep Agents v0.5 发布（Deep Agents v0.5）

LangChain 发布了 deepagents 和 deepagentsjs 的新版本 v0.5，核心亮点是支持异步（非阻塞）子 Agent，允许主 Agent 将任务委派给后台运行的远程 Agent，不再需要同步等待。此外还扩展了多模态文件系统支持，Agent 可以更灵活地处理不同类型的数据。这个更新对构建复杂 Agent 工作流的开发者来说很实用——异步子 Agent 意味着并行处理能力的提升，直接影响系统吞吐量和响应速度。对于正在用 LangChain 生态搭建 Agent 应用的团队，值得关注升级。

## 📄 研究论文

### PRAISE：基于前缀复用的 Agent 搜索训练框架（PRAISE: Prefix-Based Rollout Reuse in Agentic Search Training）

当前用强化学习训练 LLM 做多跳问答等复杂搜索任务时，存在两个痛点：长序列 rollout 计算昂贵却利用不足，且奖励只在最终答案处给出，导致严重的奖励稀疏问题。PRAISE 框架提出了一个巧妙的解法：从完整搜索轨迹中提取不同搜索轮次的前缀状态，并引入中间步骤奖励，同时实现了数据效率和信用分配的双重改进。这相当于让模型在训练过程中"复用"已有的搜索经验，而不是每次都从头来过。对于做 Agent 搜索和多跳推理的研究者来说，这个框架在方法论上有实质性贡献，综合评分也是今日最高的 10.3。

🔗 [原文链接](https://arxiv.org/abs/2604.03675) · 来源：arXiv AI · 关键词：#Agent搜索 #强化学习 #前缀复用 #多跳QA

### 字符串学序列预测 I（Stringological sequence prediction I）

这是一个计划中三篇系列论文的第一篇，代表了"组合学习计划"的首个重大突破，旨在弥合 Agent 基础理论与实际算法之间的鸿沟。论文提出了基于字符串学思想的新型序列预测算法，在时间和空间效率上表现优异，并满足与序列的字符串学复杂度度量相关的错误界。具体关注两个度量：生成序列的最小直线程序大小，以及能根据位置计算任意符号的最小自动机状态数。对 AI 安全和 Agent 基础理论感兴趣的研究者值得深读，这类工作虽然偏理论，但可能对未来 Agent 架构设计产生深远影响。

### 迈向 LLM 全自主实验室仪器控制（Toward Full Autonomous Laboratory Instrumentation Control with Large Language Models）

这项工作探索了用 LLM 和 AI Agent 来编程和自动化科学实验设备的可能性。通过一个可用作单像素相机或扫描光电流显微镜的实验装置案例，展示了 ChatGPT 如何帮助创建仪器控制脚本，大幅降低了实验定制的技术门槛。对于缺乏编程能力的实验科学家来说，这意味着他们可以通过自然语言与 AI 交互来控制复杂仪器，而不必自己写代码。这是 LLM 在科学实验自动化领域的一个实际落地案例，方向很有前景。

🔗 [原文链接](https://arxiv.org/abs/2604.03286) · 来源：arXiv AI · 关键词：#LLM #实验室自动化 #AIAgent #科学仪器控制

### Topaz：可解释的 Agent 工作流模型路由框架（Explainable Model Routing for Agentic Workflows）

现代 Agent 工作流会把复杂任务拆解为子任务，再路由到不同模型以降低成本。但现有路由架构只关注性能优化，模型能力和成本之间的权衡完全是黑箱。Topaz 框架引入了形式化的可审计性：用技能画像替代静默的模型分配，包含技能分析、成本感知路由和决策记录三个组件。这让开发者能区分"聪明的效率优化"和"预算驱动的隐性降级"。对于在生产环境中部署多模型 Agent 系统的团队，Topaz 提供了急需的透明度和可调试性。

🔗 [原文链接](https://arxiv.org/abs/2604.03527) · 来源：arXiv AI · 关键词：#模型路由 #Agent工作流 #可解释性

### Rashomon Memory：面向多视角 Agent 记忆的论辩驱动检索架构（Rashomon Memory: Towards Argumentation-Driven Retrieval for Multi-Perspective Agent Memory）

长期运行的 AI Agent 会积累服务于多个目标的经验，同一事件可能需要维护相互矛盾的解读——比如谈判中的让步，对一个目标是"信任投资"，对另一个目标是"合同负债"。现有记忆架构假设只有一种正确编码方式。Rashomon Memory 提出了一种新架构：多个目标条件化的 Agent 各自按优先级编码经验，在查询时通过论辩机制协商。每个视角维护独立的本体和知识图谱，检索时各视角提出解释并进行辩论。这个思路很有意思，为 Agent 记忆系统的设计提供了一个更贴近现实复杂性的范式。

🔗 [原文链接](https://arxiv.org/abs/2604.03588) · 来源：arXiv AI · 关键词：#Agent记忆 #多视角 #论辩驱动检索

### ActionNex：云端虚拟故障管理器（ActionNex: A Virtual Outage Manager for Cloud）

ActionNex 是一个生产级 Agent 系统，支持端到端的云故障处理，包括实时更新、知识蒸馏和基于角色与阶段的最佳下一步行动推荐。系统摄入多模态运维信号（故障内容、遥测数据、人员沟通），压缩为代表有意义状态转换的关键事件，并配合分层记忆子系统——从 playbook 蒸馏的长期 KCA 知识。对于大规模云运维团队来说，这类系统直接解决了故障处理中人工密集、依赖经验的痛点。

🔗 [原文链接](https://arxiv.org/abs/2604.03512) · 来源：arXiv AI · 关键词：#AIAgent #云运维 #故障管理 #多模态

### 迈向 AI 历史学家：从一手史料中进行 Agent 信息抽取（Towards the AI Historian: Agentic Information Extraction from Primary Sources）

研究团队推出了 Chronos 系统的首个模块，让历史学家通过自然语言交互将一手史料的图像扫描件转化为结构化数据。与固定的视觉语言模型抽取流水线不同，Chronos 允许历史学家针对异质性史料语料库自适应工作流，评估 AI 模型在特定任务上的表现，并通过自然语言迭代优化。AI 在历史研究中的应用一直受限于缺乏专为历史学家设计的工具，这项工作填补了这个空白。

🔗 [原文链接](https://arxiv.org/abs/2604.03553) · 来源：arXiv AI · 关键词：#AIAgent #历史研究 #信息抽取 #OCR

### 单 Agent vs 多 Agent：协作学习行为的自动化视频分析（Single-agent vs. Multi-agents for Automated Video Analysis of On-Screen Collaborative Learning Behaviors）

研究比较了闭源 VLM（Claude-3.7-Sonnet、GPT-4.1）和开源 VLM（Qwen2.5-VL-72B）在单 Agent 和多 Agent 设置下，对协作学习屏幕录像进行自动编码的表现。基于 ICAP 框架，提出并比较了两种多 Agent 架构。这项工作的价值在于为教育技术领域提供了 VLM 自动化分析的实证基准，同时也为单 Agent 与多 Agent 架构的选择提供了参考。

🔗 [原文链接](https://arxiv.org/abs/2604.03631) · 来源：arXiv AI · 关键词：#VLM #视频分析 #协作学习 #多Agent

### 一石六鸟：论 Agent 与代理性（To Throw a Stone with Six Birds: On Agents and Agenthood）

Six Birds Theory（SBT）将宏观对象视为诱导闭包而非原语。论文在 SBT 框架内给出了代理性的类型正确描述：一个理论诱导出具有显式接口和账本约束的层；Agent 是一个被维护的理论对象，其可行接口策略能在保持可行性的同时引导外部未来。通过有限受控系统中的四个可检验组件来操作化这一契约。这是一篇偏理论的工作，试图为"什么是 Agent"给出严格的形式化定义。

🔗 [原文链接](https://arxiv.org/abs/2604.03239) · 来源：arXiv AI · 关键词：#Agent #代理理论 #SixBirdsTheory

### 个性需要挣扎：神经进化国际象棋 Agent 中 Baldwin 效应的三种机制（Personality Requires Struggle: Three Regimes of the Baldwin Effect in Neuroevolved Chess Agents）

研究探讨了终身学习能否在进化时间尺度上扩展行为多样性。在国际象棋 Agent 中，使用 8 个 NEAT 进化的神经模块、Hebbian 游戏内可塑性和想象力信号链进行实验。结果发现一个方差交叉现象：Hebbian ON 初始方差低于 OFF，但在第 34 代反超（ρ=0.91, p<0.05），且出现三种依赖对手类型的机制。这项工作将进化生物学中的 Baldwin 效应与 AI Agent 行为多样性联系起来，视角独特。

🔗 [原文链接](https://arxiv.org/abs/2604.03565) · 来源：arXiv AI · 关键词：#神经进化 #Baldwin效应 #可塑性 #国际象棋
