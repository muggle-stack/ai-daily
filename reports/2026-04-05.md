## 📱 产品动态

### Anthropic 称 Claude Code 订阅者使用 OpenClaw 需额外付费（Anthropic says Claude Code subscribers will need to pay extra for OpenClaw usage）

Anthropic 近日宣布调整 Claude Code 的定价策略，用户在使用 OpenClaw 及其他第三方工具时将需要支付额外费用。此前 Claude Code 订阅用户可以在统一订阅价格下接入多种外部工具，但随着第三方生态的扩展，Anthropic 选择将这部分成本单独拆分。这一举措反映出 AI 编程助手市场正从"大包大揽"的订阅模式向更精细化的分层定价过渡。对于重度依赖 OpenClaw 等工具链的开发者来说，使用成本将明显上升。这也可能促使竞争对手在定价策略上做出差异化回应，值得持续关注。

### 日调用量超万亿破纪录！阿里千问3.6Plus登顶全球模型调用量榜首

阿里巴巴旗下千问3.6Plus 模型日调用量突破 **1.4万亿Token**，刷新全球大模型单日调用量纪录，正式登顶全球模型调用量榜首。这一数字意味着千问已在实际商业场景中实现了极大规模的落地应用，背后是阿里云生态中海量企业级和消费级应用的持续接入。从技术角度看，支撑万亿级日调用的推理基础设施本身就是一项重大工程挑战。这也标志着中国大模型在"调用量"这一关键商业指标上已经具备与国际顶尖模型正面竞争的实力。不过，调用量与模型质量并不完全画等号，用户场景的深度和价值密度同样值得关注。

### 19岁，常青藤辍学，这群中国年轻人重构了AI记忆

一支由多位常青藤辍学的中国年轻人组成的创业团队，打造了一套全新的 AI 记忆系统，在关键 Benchmark 上实现了现象级领先。该系统是目前**唯一原生支持指代消解（Coreference Resolution）**的 AI 记忆方案——这意味着模型能够真正理解对话中"他""那个""之前说的"等指代关系，从而实现更连贯、更精准的长期记忆。指代消解一直是长上下文对话和个性化 AI 助手中的核心难题，此前主流方案多依赖外挂模块而非原生集成。团队成员平均年龄仅19岁左右，却展现出扎实的技术功底和工程落地能力。这一案例再次说明，AI 领域的创新正越来越多地由年轻一代驱动。
