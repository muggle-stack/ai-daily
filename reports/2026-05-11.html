9.0

Katanemo Labs（DigitalOcean 旗下）发布了针对 Agentic 系统的新研究 Signals。核心问题是：构建 Agent 时产生的轨迹（traces）数量庞大，逐条人工或 LLM 审查成本极高。该方法提出一种轻量级结构化"信号"计算方式，能从实时 Agent 交互中自动筛选出最值得关注的轨迹，且不改变 Agent 的在线行为，也不需要 GPU。信号按交互、执行、环境三类模式分组，涵盖对齐偏差、停滞、脱离、失败等维度。对于正在大规模部署 Agent 的团队来说，这是一个实用的可观测性工具方向。
