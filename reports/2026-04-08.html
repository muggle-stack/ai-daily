Anthropic 的 Project Glasswing 计划——将 Claude Mythos 限制性开放给安全研究人员（Anthropic's Project Glasswing - restricting Claude Mythos to security researchers - sounds necessary to me）

8.8

Anthropic 今天没有公开发布其最新模型 Claude Mythos，而是通过全新的 Project Glasswing 计划将其限制性地提供给一小批预览合作伙伴。该模型是与 Claude Opus 4.6 类似的通用模型，但 Anthropic 声称其网络安全研究能力强大到需要给整个软件行业留出准备时间。Mythos Preview 已经发现了数千个高严重性漏洞，涵盖所有主流操作系统和浏览器。Anthropic 的判断是，随着 AI 能力快速扩散，这类能力很快会被不负责任的行为者获取，因此选择了"先防御、再开放"的策略。这是大模型厂商首次因安全能力过强而主动限制发布，标志着 AI 安全治理进入了新阶段。

Deep Agents v0.5 发布（Deep Agents v0.5）

9.3

LangChain 发布了 deepagents 和 deepagentsjs 的新版本 v0.5，核心亮点是支持异步（非阻塞）子 Agent，允许主 Agent 将任务委派给后台运行的远程 Agent，不再需要同步等待。此外还扩展了多模态文件系统支持，Agent 可以更灵活地处理不同类型的数据。这个更新对构建复杂 Agent 工作流的开发者来说很实用——异步子 Agent 意味着并行处理能力的提升，直接影响系统吞吐量和响应速度。对于正在用 LangChain 生态搭建 Agent 应用的团队，值得关注升级。

PRAISE：基于前缀复用的 Agent 搜索训练框架（PRAISE: Prefix-Based Rollout Reuse in Agentic Search Training）

10.3

当前用强化学习训练 LLM 做多跳问答等复杂搜索任务时，存在两个痛点：长序列 rollout 计算昂贵却利用不足，且奖励只在最终答案处给出，导致严重的奖励稀疏问题。PRAISE 框架提出了一个巧妙的解法：从完整搜索轨迹中提取不同搜索轮次的前缀状态，并引入中间步骤奖励，同时实现了数据效率和信用分配的双重改进。这相当于让模型在训练过程中"复用"已有的搜索经验，而不是每次都从头来过。对于做 Agent 搜索和多跳推理的研究者来说，这个框架在方法论上有实质性贡献，综合评分也是今日最高的 10.3。

Rashomon Memory：面向多视角 Agent 记忆的论辩驱动检索架构（Rashomon Memory: Towards Argumentation-Driven Retrieval for Multi-Perspective Agent Memory）

9.5
